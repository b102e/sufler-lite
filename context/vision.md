# Vision — Суфлер Lite

Суфлер Lite is a real-time AI teleprompter for stressful multilingual phone calls.

It helps immigrants in Italy survive difficult phone calls with government offices, clinics, landlords, and other organizations — by generating contextual Italian reply suggestions in real time, which the user reads aloud manually.

## Core Value

The user is always the speaker. The AI is the silent assistant behind the screen — preparing the right words before the user needs to say them.

## Target User

- Immigrants in Italy
- Non-native Italian speakers
- People facing language-barrier stress during important calls

## Emotional Goal

Reduce fear and anxiety during difficult foreign-language phone calls.
Make the user feel prepared, supported, and in control.

## What This Product Is

- A real-time AI cue-card assistant
- A call preparation + live assistance tool
- A Deepgram STT + Claude AI orchestration system

## What This Product Is NOT

- A voice assistant or AI phone bot
- A general-purpose chatbot
- A translation tool
- A language-learning app
- A TTS/voice output system
